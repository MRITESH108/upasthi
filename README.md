# UPASTHI

**UPASTHI** is a digital attendance platform to simplify the attendance system with live dashboard.  
It is an initiative under **Digital India**, aiming to modernize the college management system with technology.

## Features

- College admin with their unique id are able to access the attendance
- Easy access to student and staff data
- Supports digital management initiatives
- User-friendly interface for Admin, staff, and students

## Installation

### 1. Clone the repository

```bash
https://github.com/MRITESH108/upasthi.git
```
### 2. Initiate Server 

``` bash
cd server
npm init -y
```

### 3. Install Express and other dependencies
```bash
npm i express cors
```
